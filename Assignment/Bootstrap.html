Bootstrap

Styling webpage documents

Bootstrap is one of the most used frameworks used as front-end development. It is a framework that contains predefined CSS code. It is a framework that makes web development easy access from a mobile device too. It delivers a ready-made pattern for grids, components, and utilities. The version of Bootstrap er are using is Bootstrap 5. The framework is free to use and can be used in 2 different ways (downloading a file for including libraries in a project or directly mentioning the URL for using the online version).

Reason of Using Bootstrap

•	Uses predefined style components which makes it faster.
•	Works on a landscape device to a portrait-oriented mobile device.
•	Works on a grid system for organization and responsive page layouts.
•	Facilitates predefined components and designs for direct embedding.

Conposition and Way to Use CSS

•	container / container-fluid	gives fixed / full width while building the layout
•	row		 		creates a horizontal row group.
•	col				create responsive column (col-4, col-6)
•	m- , p- 		        Applying margin and padding (mt-3, p-2)
•	h1 to h6			create responsive text
•	text	 			used to change text property
•	list-group			used to style group items
•	table				create responsive tables
•	form-control			Used in input, text areas and in checkboxes
•	input-group			creates an input which is combined with a button or text
•	btn				Give style to button
•	navbar				used to create a responsive navigation bar
•	alert				create a feedback message
•	badge				make label indicators
•	modal				used to create pop-up dialog boxes
•	carousel			used to create a slideshow with images and manage it
•	dropdown			creates interactive menus
