JavaScript

Animating / Responding webpage content

JavaScript is a lightweight and multipurpose programming language that is mainly used for creating dynamic and interactive content on websites. JavaScript is embedded with HTML to get structure and CSS to get style. It adds activities, read-time updates, user interaction and animation on a static HTML / CSS webpage without reloading. The code written in JS is browser-independent in nature and directly understood by the browser. It is used to manipulate DOM (Document Object Model), form validation, APIs integration and building a responsive interface. It is also used in backend development after being combined with Node.js. Its ease and support build a foundation in modern web technologies.

Reason of Using Bootstrap

Roles of JavaScript:
•	Activities (clicking, typing, scrolling) of the user are handled. 
•	Update and deploy the structure of HTML. 
•	Handles the verification and validation. 
•	Update by connecting with the server without reloading the webpage. 
•	Provide dynamic content and animate it.  


How does JavaScript work?
•	Web Browser loads the HTML file and creates a Document Object Model.
•	JS code is loaded.
•	JS engine in the browser starts to run and execute code.
•	JS engine changes the DOM (like image/text/style) and responds to user action (like click/scroll)

Conposition and Way to Use CSS

Structure of JavaScript:
JavaScript is added in HTML file in three ways:
•	Inline JS:
Inline JS is the process of adding JavaScript code inside an element’s attributes.

•	Internal JS
Internal CSS is the process of adding JavaScript code inside the same HTML file in <<head>…<</head> section.

•	External JS
External JS is the process of adding JavaScript code linking an external JS file to HTML using <<script src=”script.js”> inside head section.
